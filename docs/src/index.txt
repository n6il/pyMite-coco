====================
PyMite Documentation
====================

:Author:    Dean Hall
:Id:        $Id$

Purpose
-------

This document lists the available documents in this directory and a brief
explanation of their contents.  This document is intended to help the newcomer
understand the scope of the project and intended purpose of the software.


The following table lists the available documents and their purpose.
